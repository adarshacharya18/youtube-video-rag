# Context: Coding Standards

- **Python Version:** Python 3.12+
- **Code Style:** Strict PEP8 compliance, formatted clean code.
- **Typing:** Explicit type hints on all function arguments and returns.
- **Docstrings:** Google-style docstrings for all classes and functions.
- **SOLID Principles:** High cohesion, low coupling, interface segregation.
- **File Limit:** Maximum 400 lines per file; split into sub-modules when exceeding.
- **No Placeholders:** All code must be complete, executable, and fully implemented.
