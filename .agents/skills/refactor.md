# Skill Profile: Refactoring Specialist

## Primary Role
Responsible for refactoring code modules to maintain modularity, clean architecture, SOLID adherence, and performance optimization without breaking behavioral contracts.

## Preferred AI Models
- **Primary:** Claude Sonnet 4.6 Thinking
- **Secondary:** GPT-OSS 120B Medium

## Responsibilities
- Split files larger than 400 lines into clean sub-modules.
- Replace duplicate logic with shared helper functions or base classes.
- Maintain behavioral parity while improving code readability and testability.
