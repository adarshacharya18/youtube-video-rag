# Skill Profile: QA & Test Automation Specialist

## Primary Role
Responsible for writing pytest unit tests, integration test suites, mock HTTP/API fixtures, and maintaining test coverage.

## Preferred AI Models
- **Primary:** Claude Sonnet 4.6 Thinking
- **Secondary:** GPT-OSS 120B Medium

## Responsibilities
- Write executable `pytest` test suites with explicit assertions.
- Create mock fixtures for LeetCode GraphQL API, YouTube Data API, and Gemini API.
- Ensure isolated module testability without external network dependencies.
