# .ai Workspace System Directory

This directory contains the operational AI guidelines, role-based skills, execution workflows, system context documents, and standardized prompt templates for the **Automated DSA Educational YouTube Channel** pipeline.

## Directory Structure

- **`skills/`**: Persona definitions and execution rules for specific AI engineering roles.
- **`workflows/`**: Step-by-step procedures for standard development workflows.
- **`context/`**: Core technical context, architecture specifications, standards, and roadmaps.
- **`prompts/`**: Reusable task prompts and template structures for interacting with AI models.
